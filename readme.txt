A powerful Gutenberg block to calculate and display semester-wise GPA and overall CGPA for students.

== Description ==

Dream CGPA Calculator is a comprehensive WordPress plugin that provides a beautiful, easy-to-use Gutenberg block for calculating and displaying Grade Point Average (GPA) and Cumulative Grade Point Average (CGPA).

Perfect for educational websites, university portals, student portfolios, and academic blogs.

**Features:**

* Easy-to-use Gutenberg block editor interface
* Add multiple semesters with custom names
* Add unlimited courses per semester
* Support for standard grading system (A+, A, A-, B+, B, B-, C+, C, D, F)
* Automatic GPA calculation for each semester
* Automatic CGPA calculation across all semesters
* Beautiful, responsive table display on frontend
* Real-time calculations in the editor
* Professional styling with gradient headers
* Mobile-responsive design
* Clean, modern UI matching WordPress design standards

**How to Use:**

1. Add the "Dream CGPA Calculator" block from the Gutenberg block inserter
2. Click "+ Add Semester" to add a new semester
3. Click "+ Add Course" within each semester to add courses
4. Enter Course Code, Course Title, Credit hours, and select Grade
5. The plugin automatically calculates Semester GPA and Overall CGPA
6. View the beautifully formatted table on the frontend

**Grading Scale:**

* A+ = 4.0
* A = 3.75
* A- = 3.5
* B+ = 3.25
* B = 3.0
* B- = 2.75
* C+ = 2.5
* C = 2.25
* D = 2.0
* F = 0.0

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/dream-cgpa-calculator` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress.
3. Use the block by searching for "Dream CGPA Calculator" in the Gutenberg block inserter.

== Frequently Asked Questions ==

= How do I add courses? =

Click the "+ Add Course" button within any semester to add a new course row.

= Can I edit semester names? =

Yes, you can click on the semester name in the editor to edit it.

= How is CGPA calculated? =

CGPA is calculated by taking the weighted average of all semester GPAs, where each semester's GPA is weighted by its total credits.
